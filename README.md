L'application permet de :

- Ajouter un film
- Afficher la liste des films
- Voir les détails d'un film
- Modifier un film
- Supprimer un film
- Rechercher un film par son titre
- Afficher le nombre total de films enregistrés


Technologies utilisées

- PHP 8+
- Symfony 7
- Doctrine ORM
- Twig
- MariaDB / MySQL
- HTML / CSS
